neutron

a lightweight pipeline system based on kubernetes

Enjoy neutron pipeline in 3 steps.

1

Define a pipeline with neutron.yaml. 
jobs:
  neutron-build:
    image: "golang:latest"
    trigger: ["MR","PUSH"]
    steps:
      - name: "build"
        cmd: "make cli api"

2

Register a new pipeline in neutron system.

Add neutron public key to your code base.

Create pipeline

3

Configure webhook trigger.

PUSH, MERGE and TAG are supported now.
